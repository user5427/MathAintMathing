6. Atsitiktinio dydžio ξ, i˛gyjančio neneigiamas reikšmes pasiskirstymo funkcija Fξ (x) = x3,
kai 0 ⩽ x ⩽ 1. Raskite tikimybę P (ξ < [0,5; 1]).

P(ξ < 0,5) = Fξ(0,5) = (0,5)³ = 0,125
P(ξ < 1) = Fξ(1) = (1)³ = 1
P(ξ > 1) = 1 - P(ξ < 1) = 1 - 1 = 0
P(ξ < [0,5; 1]) = P(ξ > 1) - P(ξ < 0,5) = |0 - 0,125| = 0,125
