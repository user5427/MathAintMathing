21. Dydis ξ yra geometrinis, P (ξ = 2) = 1/4. Kaip apskaičiuoti pasiskirstymo funkcijos reikšmę
Fξ (2,5)?

m = 2
P(ξ = 2) = q^(m-1) * p = 1/4
q = 1 - p
P(ξ = 2) = (1 - p)^(m-1) * p
P(ξ = 2) = (1 - p) * p = 1/4
(1 - p) * p = 1/4
p(1 - p) = 1/4
p^2 - p + 1/4 = 0
p = (1 ± √(1 - 1)) / 2
p = (1 ± 0) / 2
p = 1/2
q = 1 - p = 1/2
Fξ (2,5) = P(ξ ≤ 2,5) = P(ξ = 1) + P(ξ = 2) 
P(ξ = 1) = q^0 * p = p = 1/2
Fξ (2,5) = P(ξ ≤ 2,5) = P(ξ = 1) + P(ξ = 2) = 1/2 + 1/4 = 3/4
