21. Ivykiai A, B, C yra nepriklausomi, P (A) = P (B) = 1/2, P (C) = 1/3.
Apskaičiuokite P (A ∩ B ∩ C|A ∩ B).

P(A ∩ B ∩ C|A ∩ B) = P(A ∩ B ∩ C) / P(A ∩ B)
P(A ∩ B ∩ C) = P(A) * P(B) * P(C) = (1/2) * (1/2) * (1/3) = 1/12
P(A ∩ B) = P(A) * P(B) = (1/2) * (1/2) = 1/4
P(A ∩ B ∩ C|A ∩ B) = (1/12) / (1/4) = 1/3
