25. Paprastasis atsitiktinis dydis su vienodomis tikimybemis igyja dvi reikšmes, viena iš
reikšmiu yra 3. Kokia kita atsitiktinio dydžio reikšme, jeigu jo vidurkis lygus 5?

P(X = 3) = 0,5
P(X = x) = 0,5
E(X) = 3 * P(X = 3) + x * P(X = x)
E(X) = 3 * 0,5 + x * 0,5
5 = 3 * 0,5 + x * 0,5
5 = 1,5 + 0,5x
5 - 1,5 = 0,5x
3,5 = 0,5x
7 = x
x = 7
