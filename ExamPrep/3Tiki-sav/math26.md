26. Paprastasis atsitiktinis dydis igyja dvi reikšmes, viena iš reikšmiu yra 3, ji igyjama dvigubai
dažniau nei kita reikšme. Kokia kita atsitiktinio dydžio reikšme, jeigu jo vidurkis lygus 5?

P(X = 3) = 2/3
P(X = x) = 1/3
E(X) = 3 * P(X = 3) + x * P(X = x)
E(X) = 3 * (2/3) + x * (1/3)
5 = 3 * (2/3) + x * (1/3)
5 = 2 + (1/3)x
5 - 2 = (1/3)x
3 = (1/3)x
3 * 3 = x
x = 9
